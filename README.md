# SCT_DA_2
# Data Cleaning and Preparation using Python

## About the Project

In this project, I performed basic data cleaning and preparation on the Global Superstore dataset using Python and Pandas in Google Colab.

The main goal was to understand how raw data can be cleaned before analysis by handling missing values, removing duplicate records, converting data types, and saving the cleaned dataset for future use.

---

## Dataset Used

- Global Superstore Dataset

The dataset contains information related to orders, customers, products, sales, profit, and shipping details.

---

## Tools Used

- Python
- Google Colab
- Pandas

---

## Tasks Performed

### 1. Loaded the Dataset
Imported the dataset into Google Colab using the Pandas library.

### 2. Explored the Data
Checked the structure of the dataset and viewed sample records to understand the available columns and data types.

### 3. Identified Missing Values
Searched for empty or null values in the dataset using Pandas functions.

### 4. Handled Missing Data
Removed rows containing missing values to improve data quality.

### 5. Removed Duplicate Records
Checked for duplicate rows and deleted them to avoid incorrect analysis results.

### 6. Converted Data Types
Converted the **Order Date** column from text format to datetime format for easier analysis.

### 7. Exported the Cleaned Dataset
Saved the cleaned dataset as a new CSV file.

---

## Output

The final cleaned dataset was exported as:

**Cleaned_Global_Superstore.csv**

The cleaned data:
- Contains no duplicate records
- Has missing values handled
- Includes properly formatted date columns
